# Combinations

## Task
- Given two integers n and k, return all possible combinations of k numbers out of 1 ... n.

Example:

Input: n = 4, k = 2
Output:
[
  [2,4],
  [3,4],
  [2,3],
  [1,2],
  [1,3],
  [1,4],
]

## Understanding
 Input
  - Two integers
  - n and k
Output
  - Array
    + Nested arrays of k elements

## Examples
    
1, 2, 3, 4

         1  - 4    2 - 4
         | \        \
         2  3         3
